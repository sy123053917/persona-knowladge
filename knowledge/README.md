## 📚 目录结构

```
knowledge/
├── 01-每日笔记/       # 日常记录、工作日志
├── 02-项目记录/       # 具体项目/任务的背景、进度、决策
├── 03-技能库/         # 技能文档、经验总结
├── 04-游戏攻略/       # 仗剑传说等游戏攻略
├── 05-技术文档/       # 技术研究、文章、代码片段
└── 06-收藏资源/       # 有价值的链接、参考资料
```

## 🔑 核心文件

| 文件 | 作用 | 更新频率 |
|------|------|----------|
| `MEMORY.md` | 长期记忆精华 | 每周整理 |
| `AGENTS.md` | AI助手指南 | 按需更新 |
| `SOUL.md` | AI人格设定 | 初始配置 |

---

## 💡 使用方法

### 日常记录
每天把重要信息写入 `01-每日笔记/YYYY-MM-DD.md`

### 项目管理
每个项目创建 `02-项目记录/项目名/PROJECT.md`

### 技能积累
将经验写入 `03-技能库/SKILL-NAME.md`

### 检索
告诉AI："帮我查一下关于xxx的记忆"
